# 機能概要
PHP/MySQL の開発環境構築
# 使用方法

## 対話モード
```zsh
docker exec -it php_php_1 bash
root@5582db8264c1:/var/www/html# php -a
```
# 動作環境
windows, intel mac
# 動作環境の構築方法
```zsh
docker-compose up -d
```

PHPの動作チェックのため http://localhost:8080/ にアクセスします。

MySQL & phpMyAdmin の動作チェックのため http://localhost:4040/ にアクセスします。
